前言
--------
ecard 电子贺卡服务平台
用来在线发送自定义的电子贺卡


安装要求
--------
- linux or ubuntu
- python 2.6 or 2.7
- django (1.5+)

安装必要包
--------

- 1.download:

from pypi::

    $ pip install ecard 

or::

    $ easy_install ecard 

or clone from github::

    $ git clone https://github.com/wcc526/ecard.git

- 2.必要的包:: 

    django_admin_bootstrapped 
    bootstrap_toolkit
    apt-get install imagemagick  
    apt-get install python-pythonmagick  

enjoy it!
